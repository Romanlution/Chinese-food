# 中式面点刷题

中式面点考试刷题网页，面向手机使用。题库来自 `中式面点知识点.docx`，当前包含 599 条有效知识点，并自动生成 100 道单选题练习。

## 功能

- 每套 100 道单选题
- 选项强关联题干焦点，尽量避免无意义干扰项
- 点击「下一题」时判断对错
- 答错自动加入错题库
- 错题连续答对 3 次后转入归档
- 错题库和归档保存在浏览器本地
- 支持 PWA，可添加到手机主屏幕并离线使用

## 本地运行

```bash
python3 -m http.server 4173
```

打开：

```text
http://127.0.0.1:4173/
```

同一 Wi-Fi 下，手机可访问电脑局域网地址。

## Vercel 部署

这是纯静态项目，不需要构建命令。部署时：

- Framework Preset: `Other`
- Build Command: 留空
- Output Directory: 留空
- Root Directory: 仓库根目录

部署后用手机浏览器打开 Vercel 地址，iPhone 可通过 Safari 分享菜单选择「添加到主屏幕」。
